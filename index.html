About Me

Hi, I'm Anrika Schwartz and this is my website!

I was born on June 2nd, 1926 in Galați, Romania. When me and my husband got merried we moved to Brăila, his home town.

Later on we moved to Bucharest and lived there until we immigrated to Israel in 1962 and setelled in Kiryat Shmona.

A few years later we moved to Tel Aviv and currently I'm leaving in Or Yehuda.

I raised a beautiful family: two daughters, six grandchildren and seven great-grandchildren, with whom I love to spend my time.

My hobbies are: Listening to music in French and Romanian, knitting and looking in Pinterest for new patterns and knitting styles,
reading books on my Kindle and solving puzzles online.

Q&A

  1. The main differences between Web 1.0, 2.0 and 3.0:


    Web 1.0:
    The “readable” part of the World Wide Web with flat data. In Web 1.0, there is only a limited interaction between sites and web users. Web 1.0 is simply an information portal where users passively receive information without being given the opportunity to post reviews, comments, and feedback.

    Web 2.0
    The “writable” part of the World Wide Web with interactive data. Unlike Web 1.0, Web 2.0 facilitates interaction between web users and sites, so it allows users to interact more freely with each other. Web 2.0 encourages participation, collaboration, and information sharing. Youtube, Wiki, Flickr, Facebook are considered to be Web 2.0 applications.

    Web 3.0
    The “executable” part of Word Wide Web with dynamic applications, interactive services, and “machine-to-machine” interaction. Web 3.0 is a semantic web which refers to the future. In Web 3.0, computers can interpret information like humans and intelligently generate and distribute useful content tailored to the needs of users. One example of Web 3.0 application is Tivo, a digital video recorder. Its recording program can allow you to search the web for things to what and read you out load what it finds, based on your preferences.
